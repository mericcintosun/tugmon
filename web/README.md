# Tugmon web (Next.js)

This directory is the Next.js front end and API routes for **Tugmon**.

- **Run:** `npm install && npm run dev`
- **Env:** copy `.env.example` to `.env.local` (see repo root `README.md` for full variable table).
- **Deploy:** build with `npm run build`; set `NEXT_PUBLIC_*` and server secrets on your host.

The repository overview, contract setup, and PRD differences are documented in the [**root README**](../README.md).
